Procedure

From start to finish


Step 1

Obtain Materials:

  • 10x 100mm Petri Dishes
  • Luria-Bertani agar (LB agar)
  • Tube of Pseudomonas Fluorescens
  • Tube of F. Oxysporum
  • Tape
  • 15x innoculating loops
  • ImageJ online software

Step 2

Set up the experiment:

  1. Pour the LB agar into each of the petri dishes and let it dry
  2. Import the F. Oxysporum using innoculating loops
  3. Wait 3 days
  4. Import the Pseudomonas Fluorescens using innoculating loops
  5. Seal the agar plate using tape

Step 3

Collect and Analyze Data

  1. Take pictures of each of the agar dishes
  2. Import images into ImageJ software
  3. Use the polygon tool to find the funcal colony areas
  4. Find the average of the colony areas
  5. Repeat this every 7 days for 4 weeks
